💡 Consejos para Mejores Prompts

🎯 Sé Específico

Define claramente qué quieres, para quién y con qué formato.

📏 Define Límites

Especifica longitud, tono, nivel técnico y elementos requeridos.

🔧 Proporciona Contexto

Incluye información relevante sobre el proyecto o situación.

📋 Usa Ejemplos

Muestra el formato o estilo que esperas en la respuesta.

🔄 Itera y Mejora

Refina tus prompts basándote en las respuestas obtenidas.

⚡ Mantén la Claridad

Usa lenguaje claro y evita ambigüedades en tus instrucciones.
